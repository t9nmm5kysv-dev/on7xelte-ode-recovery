# Public repository scope

This public repository is a research write-up and tooling reference.

It intentionally excludes:

- Samsung proprietary binaries
- patched Samsung binaries
- generated password candidate lists
- phone-side cache dumps
- raw recovery working directories
- recovered data
- found credentials

The public repo is intended to document the method, evidence, limitations, and reproducibility requirements without publishing device-specific proprietary artifacts or password-pattern material.
